Projects

Crypto Bank

Written with a group in python, this program models a secure banking and ATM system. A user interacts with an client ATM that communicates with a bank server. This program made use of a simipified SSL protocol, and several cryptographic algorithms including; AES, Pallier, RSA, SHA1, and HMAC.

Wordle Server

Written in C, this program uses client and server programs to model the NYT Wordle game. A user connects to the server via the client program, a word is randomly selected from the dictionary file, and the user then has 6 guesses to correctly determine the word. Program makes use of socket communication.

Best Professor List

Group project written in python, JavaScript React, and CSS. Intended to be used by RPI students, this website congregates RPI professor data scraped from RateMyProfessor.com in one place. Students can search for a professor and view their rating, or look up an RPI course code to see the highest rated professors that teach it. To develop this project, we implemented Agile scrum methedology and had weekly sprints.
